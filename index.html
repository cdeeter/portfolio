About Me

Hello! I'm a front-end web developer located in Oakland, California.

I develop responsive websites and applications that look appealing in all devices, from desktop computers to mobile devices!

I currently code in HTML, CSS, and JavaScript (including jQuery and AJAX) and plan to expand my skillset with other coding languages in the future.
